# Transportation Representative Resume App

This is a Flutter application that serves as an interactive resume tailored for a Transportation Representative position, specifically matching the responsibilities and qualifications of the Amazon L2 associate role.

## Features
- **Profile Section:** Highlights relevant communication and problem-solving skills.
- **Experience Section:** Showcases logistics background, issue troubleshooting, and process improvement.
- **Skills Section:** Lists relevant technical and soft skills including Excel, typing, email applications, and customer service.
- **Education Section:** Highlights graduation status.
- **Responsive Design:** Works on mobile, tablet, and desktop screens.

## Tech Stack
- Flutter
- Dart

## Setup
1. Clone the repository.
2. Run `flutter pub get` to fetch dependencies.
3. Run `flutter run` to launch the application.
